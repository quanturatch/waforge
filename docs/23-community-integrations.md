# Community Integrations

Third-party integrations and adapters built by the community on top of WaForge's REST API.

> ⚠️ **These projects are community-maintained and are not affiliated with or endorsed by WaForge.**
> They may lag behind WaForge API changes. Verify compatibility and review the source before using
> them in production.

| Integration | Platform | Source | Notes |
| --- | --- | --- | --- |
| **ioBroker.WaForge** | [ioBroker](https://www.iobroker.net/) (home / IoT automation) | [ThorstenBoettler/ioBroker.WaForge](https://github.com/ThorstenBoettler/ioBroker.WaForge) · [npm](https://www.npmjs.com/package/iobroker.WaForge) | Send text / image / video / audio / document messages to chats and groups through WaForge's REST API, with Blockly blocks. Early stage. |

Maintain an integration built on WaForge and want it listed here? Open a PR adding a row (or an issue), including the repository link and a one-line description.

For the first-party n8n nodes, see [n8n Integration](./22-n8n-integration.md).
